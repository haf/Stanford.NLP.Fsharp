Stanford.NLP.Fsharp

F# extentions for Stanford.NLP.NET

Available packages

The Stanford.NLP.Parser.Fsharp library can be installed from NuGet: 
PM> Install-Package Stanford.NLP.Parser.Fsharp

Content

Stanford.NLP.Parser.Fsharp provides:

  • Strongly-typed set of Penn Treebank II tags.

  • Active patterns for tags classification.

  • Helper functions for java.util.Iterable.
